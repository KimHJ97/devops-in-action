# DevOps In Action

DevOps와 Infra 관련 내용을 정리한다.

 - Docker: 도커 관련 내용 정리
 - Infra: 인프라 및 서버 관련 내용 정리
 - Kubernetes: 쿠버네티스 관련 내용 정리
 - Deploy: 빌드 및 배포(CI/CD) 관련 내용 정리
 - AWS: AWS Cloud 관련 내용 정리
 - ../_Lecture 폴더: 관련 강의 정리
 - ../_Books 폴더: 관련 도서 정리
 - ../_Concept 폴더: 관련 개념 정리
 - ../_Usage 폴더: 관련 상황별 사용 예제 정리
